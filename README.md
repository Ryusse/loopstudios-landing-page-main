**<h2>Setup the project</h2>**

  * Fork the project

  * Clone this project in the folder of your choice

    ```bash
    git clone https://github.com/Ryusse/loopstudios-landing-page-main.git
    ```

  * Install the dependencies

  * If you use npm 

    ```bash
    npm install
    ```

  * If you use yarn

    ```bash
    yarn install
    ```

  * Run the server

    ```bash
    npm run dev
    ```
